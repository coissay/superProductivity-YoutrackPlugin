⚙️ Automatic sprint ({CURRENT_SPRINT})
If disabled, {CURRENT_SPRINT} is not substituted in the query (useful to avoid clashing with a sprint filter already hardcoded).
Placeholders: {YEAR} (2026), {YEAR_SHORT} (26), {SPRINT_NUM} (8), {SPRINT_NUM_PADDED} (08) — e.g.: 26S08

✅ Status handling
Comma-separated YouTrack statuses (case-insensitive). Matching tickets are marked Done in SuperProductivity instead of getting a status tag — plugins can't archive tasks directly, so this is the closest equivalent; SuperProductivity's own archiving takes over from there.
E.g. a ticket removed from the current sprint. Only affects tickets seen in a previous sync — has no effect the first time you enable it. An empty sync result is ignored (treated as a likely misconfigured query) so it never marks everything Done at once. Be careful if you frequently change the query/sprint filter, since tickets that simply fall outside the new query will also get marked Done.
🛠️ Field mapping
The name of the YouTrack custom field holding the due date, exactly as it appears in your instance (case-sensitive). Defaults to "Due Date".
